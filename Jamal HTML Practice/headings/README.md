```html
<!DOCTYPE html>
هذا السطر يخبر المتصفح أن الصفحة تستخدم إصدار HTML5.

<html>
هذا هو العنصر الأساسي الذي يحتوي كل محتويات الصفحة.

<head>
يحتوي معلومات غير ظاهرة للمستخدم مثل العنوان والترميز.

<meta charset="UTF-8">
يسمح بظهور اللغة العربية والرموز بشكل صحيح داخل الصفحة.

<title>
هو اسم الصفحة الذي يظهر في أعلى المتصفح داخل التبويب.

<body>
كل ما يراه المستخدم داخل الصفحة يكون داخل هذا العنصر.

<h1> إلى <h6>
هذه عناصر العناوين في HTML.
تستخدم لتنظيم المحتوى حسب الأهمية.

<h1> هو العنوان الرئيسي والأكبر.
<h2> عنوان فرعي تحت h1.
<h3> عنوان داخل قسم h2.
وهكذا حتى <h6> وهو أصغر مستوى عنوان.

عادةً نستخدم 
h1
 مرة واحدة فقط في الصفحة لأنه يمثل الفكرة الرئيسية.

ثم
 نستخدم 
 h2 و h3
لتنظيم الأقسام مثل فصول كتاب.

```
